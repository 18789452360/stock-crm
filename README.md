# Mr.Stock.CRM

### 预览项目

1. 安装 `http-server`
```bash
npm i http-server -g
```

2. 进入到项目目录, 然后启用项目
```bash
cd Mr.Stock.CRM
http-server .
```

3. 访问地址
```bash
http://127.0.0.1:8080
```
